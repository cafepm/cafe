Qué es XML ?

  • XML : eXtensible Markup Language

  • Qué es XML

    • Es un subset de SGML ...
    • que describe una clase de objetos de datos llamados “documentos XML”

  • Qué es SGML ??

    • Método para describir formalmente un lenguaje (metalenguaje)
    • ... un lenguaje electrónico de texto con mark-up (colección de señas -tags- que describen las especificaciones de un documento electrónico, por ejemplo para darle formato)
