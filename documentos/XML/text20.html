XML::Simple

  • Una herramienta todo en uno para XML

    • Provee lectura, parsing y escritura de XML
    • Utiliza stream parsing (XML::Parser, XML::SAX, etc.)

  • De uso simple :-)

  • Traslada al XML en una estructura Perl : aquí es donde se encuentra el gran problema :

    • devuelve un hashref
    • los keys del hash son los distintos tags reconocidos
    • cada atributo de un tag se accede como una key de un hash
    • los elementos repetitivos se acceden como arrays (configurable)
