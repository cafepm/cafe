XML::Simple (limitaciones)

  • Hay ciertas limitaciones que podrían perjudicarnos :

  • No se toma en cuenta el texto que contiene sólo espacios

  • Al poner todo en un hash el orden de los tags se pierde

  • No hay un control preciso sobre el formato de salida

  • Las keys de un hash deben ser nombres de elementos de XML acordes a la especificación

  • La conversión de codificación de caracteres no está incluida

  • Pero también hay muchos flags de XML::Simple que modifican el comportamiento de XMLin y XMLout
