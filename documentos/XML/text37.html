CaFe.pm Perl Mongers de Capital Federal

  • Perl Mongers (http://www.pm.org/)

  • Asociación internacional de grupos de usuarios Perl

  • CaFe.pm (http://cafe.pm.org/)

  • Perl Mongers de Capital Federal tiene como tarea la promoción de Perl en Capital Federal a través de ayudas, discusión de temas técnicos, compartir información, etc.

  • Lista de correo activa ( http://mail.pm.org/mailman/listinfo/cafe-pm )

  • Reuniones mensuales ( http://cafe.pm.org/reuniones/ ) :

  • primer Viernes de cada mes, después de un Miércoles, a las 19:08

  • Reviews de libros ( http://cafe.pm.org/reviews/ )

  • CaFe Perl, publicación mensual ( http://cafe.pm.org/boletines/ ) : secciones fijas, código, explicaciones detalladas, entrevistas, noticias.

  • Canal de IRC : participamos en #perl-es en irc.freenode.org
