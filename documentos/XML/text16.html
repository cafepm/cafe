Parsing (2)

  • Hay Distintas formas de modelar el uso/acceso de documentos XML ...

  • como objetos (DOM – Standard)

    • objetos : parte importante de recursos para administrar interfaz
    • árbol : toda la estructura DOM en memoria
  • como un filesystem (XPath – Standard)
    • más simple : no son objetos
  • como un stream o serie ordenada de elementos simple, implementado por eventos (Expat, SAX)
    • stream : cada elemento se trata una sola vez
    • eventos : implica un modelo desacoplado (starttag, etc.)
    • el scope es local a cada elemento : necesita pocos recursos
