XML::SAX (Parsing)

  • El método es con callbacks (similar a XML::Parser), pero ...

  • Interfaz OO

    • Mayor flexibilidad y adaptación
    • Independencia de la implementación
    • Herencia / fallback

  • Usa XML::SAX::ParserFactory

    • XML::Parser como default parser
    • Se puede agregar otros parsers

  • Mayor coherencia

    • Sólo se establece la clase (package) que recibe todas las callbacks
    • Las callbacks sólo reciben un hashref (en XML::Parser varía con cada tipo de llamada)
