... y DTD ?

  • DTD : Document Type Declaration

    • Es un documento XML ...
    • que provee la gramática particular para cada documento (relación entre los elementos de la sintaxis genérica de XML)
    • XML provee la sintaxis (formas válidas de combinación de los elementos)

  • <?xml version="1.0" encoding="utf-8" ?>

  • <!ELEMENT Survey (name,Answers)>

  • <!ATTLIST Survey id>

  • <!ELEMENT name (#PCDATA)>

  • <!ELEMENT Answers (#PCDATA)>

  • <!ATTLIST Answers nQuest>
