Mini-CPAN

  • Cómo hago para instalar módulos en un equipo sin acceso a Internet ?? ... instalando un CPAN propio !!

  • 1.- instalar el módulo CPAN::Mini

  • # cpan

  • cpan> install CPAN::Mini

  • 2.- obtener la dirección del mirror más cercano

  • # cpan

  • cpan> o conf urllist

    • urllist
      • ftp://mirrors.bannerlandia.com.ar/mirrors/CPAN/

  • o desde http://www.cpan.org/SITES.html
