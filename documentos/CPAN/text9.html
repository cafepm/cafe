CPAN - instalación

  • Ya lo elegí (File::Temp) ... cómo lo instalo ??

  • Automatizada

    • script cpan
    • interfaz de comandos (install File::Temp)
    • usa el módulo CPAN.pm

  • Artesanal

    • download (browser, wget http://..., etc.)
    • perl Makefile.PL
    • make
    • make test
    • make install
