Cómo usarlos

La documentación que se encuentra aqui es de uso público y ha sido generada por nosotros para la libre difusión de Perl. Como tal, los distintos documentos pueden ser descargadas de este site y usadas como parte de un plan de educación, para la capacitación personal con una visualización a través de este site, o bien como base para otras tareas.

Cabe destacar que estos documentos se hayan licenciados bajo GNU GPL v2, a menos que se especifique otro tipo de licenciamiento.
